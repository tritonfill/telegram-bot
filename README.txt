
# Telegram Translator Bot

## Как запустить

1. Установи Python (если не установлен)
2. Установи зависимости:
    pip install python-telegram-bot googletrans==4.0.0-rc1

3. Запусти бота:
    python bot.py

## Возможности:
- Перевод текста на английский, русский, немецкий.
- Автоматическое определение исходного языка.
- Удобный выбор языка через кнопки.
